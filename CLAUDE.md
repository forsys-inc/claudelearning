# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is an Obsidian-based knowledge wiki for studying the **Claude Certified Architect – Foundations** certification exam. The wiki covers 5 exam domains with detailed notes, examples, and practice questions.

## Repository Structure

- `/wiki/` — Obsidian vault containing all study materials
  - `00 - Home.md` — Main landing page and learning path
  - `01 - Agentic Architecture/` — Domain 1 (27% of exam)
  - `02 - Tool Design & MCP/` — Domain 2 (18% of exam)
  - `03 - Claude Code Configuration/` — Domain 3 (20% of exam)
  - `04 - Prompt Engineering/` — Domain 4 (20% of exam)
  - `05 - Context Management/` — Domain 5 (15% of exam)
  - `Sample Questions/` — Practice questions with explanations
  - `Exercises/` — Hands-on preparation exercises

## Obsidian Conventions

- Internal links use `[[Page Name]]` wikilink syntax
- Tags use `#domain/1`, `#domain/2`, etc. for domain filtering
- Each topic file follows the template: concept overview, key knowledge points, skills, 3-4 examples, and links to related topics
- Frontmatter uses YAML with `tags`, `domain`, and `weight` fields

## Content Source

All content is derived from the Claude Certified Architect – Foundations Certification Exam Guide PDF in the repo root.
